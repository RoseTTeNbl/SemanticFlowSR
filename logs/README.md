# Logs

Top-level historical run logs are not part of the current evidence surface.

Current experiment logs should either be written inside the corresponding
result directory or removed after the summary/result files are verified.

```text
results/clean_benchmark_20260701/ablations/complete_expression_semantic_fm_20260707/runs/<tag>/
```

Do not leave failed smoke logs under `logs/complete_expression_semantic_fm/`
unless they are actively being monitored.
